Detailed Description

This (very important) abstract class represents the root behaviour that will define all the properties of the CharacterActor. It can be also defined as the main interface between the core and the implementation parts. For instance, the CharacterStateController is a CharacterActorBehaviour component. The CharacterActor component will execute its CharacterActorBehaviour instance before anything else.

Implement this abstract class with your own behaviour code to define the character.
